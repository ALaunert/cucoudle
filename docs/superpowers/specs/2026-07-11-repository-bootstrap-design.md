# Cucoudle: первоначальная структура репозитория

## Назначение

Cucoudle — мобильное приложение для удалённого управления сессиями Cursor, Codex и Claude.

## Объём первого шага

На этом этапе создаётся приватный GitHub-репозиторий `ALaunert/cucoudle` с основной веткой `main` и кратким README. Репозиторий фиксирует назначение проекта, но не навязывает технологический стек, архитектуру или неподтверждённые функции.

## Структура

Первый коммит содержит ровно два файла:

- `README.md` — название, краткое описание и текущий статус проекта;
- `docs/superpowers/specs/2026-07-11-repository-bootstrap-design.md` — эта спецификация, единственная в первоначальном репозитории.

Другие файлы в первый коммит не входят.

## Публикация

Локальная ветка `main` публикуется в приватный репозиторий GitHub. Точное описание репозитория: «Мобильное приложение для удалённого управления сессиями Cursor, Codex и Claude».

## Проверка

После публикации проверяется, что:

- репозиторий доступен владельцу и имеет видимость `PRIVATE`;
- ветка по умолчанию — `main`;
- дерево первого коммита содержит ровно два файла, перечисленных выше;
- README и спецификация описывают мобильное приложение для удалённого управления сессиями Cursor, Codex и Claude;
- README и спецификация не вводят технологический стек, архитектуру или дополнительные функции;
- GitHub-описание точно совпадает с указанным выше текстом.
